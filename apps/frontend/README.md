# AI测试管理平台 - 前端

## 技术栈
- Vue 3
- TypeScript
- Vite
- Pinia
- Element Plus
- Vue Router

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 项目结构

```
frontend/
├── src/
│   ├── views/      # 页面组件
│   ├── router/     # 路由配置
│   ├── App.vue     # 根组件
│   └── main.ts     # 入口文件
├── index.html
├── package.json
├── vite.config.ts
└── tsconfig.json
```
