# 🤖 AI测试管理平台

一个让测试工作更轻松的智能管理系统！

## 快速开始（小白必看）

👉 **[点击查看详细文档](./docs/README.md)**

### Docker一键启动

```bash
cd docker
docker-compose up -d
```

然后打开浏览器访问：**http://localhost:80**

### 登录信息
- 账号：admin
- 密码：123456

---

## 主要功能

- ✅ 用户登录认证
- ✅ 工作台（统计数据）
- ✅ 项目管理
- ✅ 测试用例管理（支持AI生成）
- ✅ 缺陷管理
- ✅ 需求管理
- ✅ 策略管理
- ✅ AI中心

---

## 项目架构

```
前端 (Vue3 + TypeScript)
    ↓
后端微服务 (Spring Boot)
    ↓
MySQL + Redis
```
