# AI测试管理平台 - 本地部署说明

## 当前状态

✅ **数据库已就绪** - 本地MySQL数据库已初始化完成

## 环境要求

- **JDK 17+** - 后端Java运行环境
- **Maven 3.8+** - 后端项目构建工具
- **Node.js 16+** - 前端运行环境
- **MySQL 8.0+** - 数据库（端口3306）
- **Python 3.8+** - 辅助工具（已使用）

## 快速启动

### 1. 数据库配置
确保本地MySQL正在运行：
- 端口：3306
- 用户名：root
- 密码：root
- 数据库名：ai_test_platform

### 2. 启动后端服务

在项目根目录运行：
```bash
start-backend.bat
```

这会按顺序启动以下服务：
- Auth Service (端口: 9001)
- User Service (端口: 9002)
- Project Service (端口: 9003)
- Requirement Service (端口: 9004)
- Testcase Service (端口: 9005)
- Defect Service (端口: 9006)
- Dashboard Service (端口: 9007)
- Strategy Service (端口: 9008)

### 3. 启动前端服务

在新的命令行窗口运行：
```bash
start-frontend.bat
```

前端会在 http://localhost:5173 启动
