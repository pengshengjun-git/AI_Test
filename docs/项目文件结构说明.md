# AI测试管理平台 - 项目文件结构说明

---

## 一、项目概述

**项目名称**：AI测试管理平台  
**项目定位**：企业级智能化测试全流程管理平台，依托AI能力实现测试自动化、智能化、标准化  
**核心目标**：AI自动生成测试用例、接口自动化测试、UI自动化测试、智能回归测试、缺陷智能分析、测试报告自动生成

---

## 二、项目目录结构

```
AI-Test-Platform/                              # 项目根目录
├── .trae/                                     # Trae IDE配置目录
│   ├── agents/                                # AI角色定义（23个角色）
│   ├── rules/                                 # 项目规则配置
│   └── specs/                                 # 功能规格说明
├── apps/                                      # 应用代码目录
│   ├── frontend/                              # 前端应用（Vue3 + TypeScript）
│   ├── backend/                               # 后端微服务（Spring Boot）
│   └── ai-service/                            # AI服务（LLM调用）
├── services/                                  # Docker服务配置
│   ├── *.Dockerfile                           # 各服务Docker镜像配置
│   └── docker-compose*.yml                    # Docker Compose编排
├── deploy/                                    # 部署脚本
├── config/                                    # 配置文件
├── docs/                                      # 项目文档
├── memory/                                    # 项目记忆系统
├── logs/                                      # 日志目录
└── [根目录配置文件]
```

## 五、快速启动指南

### 5.1 Docker一键启动

```bash
# 开发环境
cd services
docker-compose -f docker-compose.dev.yml up -d --build

# 生产环境
cd deploy
docker-compose up -d --build
```
