#### 问题1：项目创建接口异常 ✅ 已修复

- **问题ID**: BUG-001
- **严重程度**: 🔴 高
- **优先级**: P0
- **问题模块**: 项目管理
- **问题描述**: 创建新项目时，点击"确定"按钮后，前端显示"服务器内部错误，请稍后重试"
- **复现步骤**:
  1. 登录系统
  2. 进入项目管理页面
  3. 点击"新建项目"
  4. 填写项目名称
  5. 点击"确定"
- **问题定位**:
  - 前端传递状态值（PLANNING/IN_PROGRESS/COMPLETED/ARCHIVED）与后端期望值（active/archived）不匹配
- **修复方案**: 在 `ProjectServiceImpl.java` 中添加状态值映射方法，将前端状态值统一映射为后端状态值
- **修复状态**: ✅ 已修复
- **修复人**: 后端开发工程师
