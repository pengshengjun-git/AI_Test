# AI测试管理平台 - 技术文档

## 1. 项目概述

### 1.1 项目简介
AI测试管理平台是一个基于AI驱动的软件质量工程平台，旨在提升测试效率和质量，支持测试用例自动生成、AI缺陷分析、智能测试策略等功能。

### 1.2 项目目标
- 实现测试用例的AI自动生成
- 提供智能化的缺陷分析能力
- 支持自动化测试策略管理
- 提供可视化的测试报告和数据分析

## 9. 操作指南

### 9.1 本地开发启动

**前端：**
```bash
cd frontend
npm install
npm run dev
```

**后端（每个服务）：**
```bash
cd backend/{service-name}
mvn spring-boot:run
```

### 9.2 Docker启动

```bash
# 进入docker目录
cd docker

# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down

# 重启服务
docker-compose restart
```

### 9.3 服务访问

| 服务 | 访问地址 |
|------|----------|
| 前端 | http://localhost |
| 认证服务 | http://localhost/api/v1/auth |
| 项目服务 | http://localhost/api/v1/projects |
| 健康检查 | http://localhost/api/v1/{service}/health |

### 9.4 迁移到云服务器

**阿里云/腾讯云部署步骤：**

1. **创建服务器**
   - 选择CentOS 7/8或Ubuntu 20.04+
   - 配置安全组，开放端口：80, 443, 3306, 6379

2. **安装Docker**
   ```bash
   # CentOS
   yum install -y docker
   systemctl start docker
   systemctl enable docker
   
   # Ubuntu
   apt-get update
   apt-get install -y docker.io
   systemctl start docker
   systemctl enable docker
   ```

3. **上传代码**
   ```bash
   scp -r ai-test-platform root@your-server-ip:/opt/
   ```

4. **启动服务**
   ```bash
   cd /opt/ai-test-platform/docker
   docker-compose up -d
   ```

5. **配置域名（可选）**
   - 在云服务商控制台配置域名解析
   - 更新Nginx配置添加SSL证书
