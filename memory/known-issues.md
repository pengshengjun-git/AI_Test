## 常见问题FAQ

### Q1: 如何启动项目？
**A**: 
```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# 前端
cd frontend
npm install
npm run dev
```

### Q2: 数据库连接失败怎么办？
**A**: 
1. 检查SQLite数据库文件是否存在
2. 检查数据库路径配置（.env文件）
3. 重新初始化数据库：`python init_db.py`

### Q3: 前端页面无法访问？
**A**: 
1. 检查后端API服务是否启动
2. 检查CORS配置是否正确
3. 检查前端API代理配置

### Q4: AI角色不工作？
**A**: 
1. 检查AGENTS.md文件是否存在
2. 检查agents目录下的角色定义文件
3. 查看控制台错误日志
